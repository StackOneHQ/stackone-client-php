# TimeOffBalancesValue

The unified value for the duration unit of the time off balance. If the provider does not specify this unit, the value will be set to unknown


## Values

| Name      | Value     |
| --------- | --------- |
| `Minutes` | minutes   |
| `Hours`   | hours     |
| `Days`    | days      |
| `Weeks`   | weeks     |
| `Months`  | months    |
| `Years`   | years     |
| `Unknown` | unknown   |
